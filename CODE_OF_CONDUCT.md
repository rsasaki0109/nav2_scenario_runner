# Code of Conduct

This project aims to be a practical, technical, and welcoming space for Nav2 scenario testing work.

## Expected Behavior

- Be respectful and direct.
- Keep technical criticism focused on code, design, tests, and evidence.
- Assume contributors have different robotics, ROS 2, simulator, and CI backgrounds.
- Prefer reproducible examples over vague claims.
- Help maintainers keep issues and pull requests actionable.

## Unacceptable Behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing private information without permission.
- Repeatedly derailing technical discussions.
- Abusive behavior in issues, pull requests, chats, or community forums.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or limit participation when behavior harms the project. Serious issues should be reported privately to the maintainers once the repository hosting configuration supports private reporting.
